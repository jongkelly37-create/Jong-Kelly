# Jong Kelly - Portfolio Website

A modern, responsive personal portfolio website showcasing skills, projects, and professional experience.

## Features

- **Responsive Design**: Optimized for mobile, tablet, and desktop
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: Hover effects, smooth scrolling, and filtering
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Accessibility**: Alt text for images and semantic structure
- **Fast Loading**: Optimized images and efficient code

## Pages

- **Home**: Landing page with introduction and call-to-action
- **About**: Personal background, career goals, and education
- **Resume**: Work experience, skills, and downloadable CV
- **Portfolio**: Showcase of projects with filtering
- **Testimonials**: Client feedback and ratings
- **Contact**: Contact form and social media links

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Responsive Design

## Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Customize the content with your personal information
4. Replace placeholder images in the `images/` folder
5. Update contact information and social media links

## Customization

- **Colors**: Modify the color scheme in `css/style.css`
- **Content**: Update text and information in the HTML files
- **Images**: Replace placeholder images with your own photos
- **Projects**: Add or modify project cards in `portfolio.html`
- **Resume**: Update work experience and skills in `resume.html`

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
